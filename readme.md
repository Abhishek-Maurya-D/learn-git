| Command | Function |
|--------|----------|
| `git init` | Initialize a new Git repository in a folder |

| `git add <file>` | Stage changes for the next commit |
| `git commit -m "msg"` | Save staged changes with a message |
| `git push` | Upload commits to a remote repository |
| `git pull` | Fetch and merge latest changes from remote |
| `git status` | Show current repo status and tracked/untracked files |
| `git checkout <branch>` | Switch to another branch |
| `git branch` | List or create branches |
| `git merge <branch>` | Merge another branch into the current branch |
| `git rebase <branch>` | Reapply commits on top of another base branch |
| `git remote -v` | Show connected remote repositories |
| `git fetch` | Download changes from remote without merging |
| `git stash` | Temporarily save uncommitted changes |
